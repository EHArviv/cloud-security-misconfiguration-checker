# Project Notes

This project demonstrates static cloud security misconfiguration detection using Python.

The tool reads demo AWS-style JSON configuration files and identifies common cloud security risks.

## Main Goals

- Practice cloud security review logic
- Detect common misconfigurations
- Generate structured security reports
- Demonstrate Security Engineer and Cloud Security workflows
- Avoid using real cloud accounts, credentials, or production data

## Safety

This project does not connect to AWS, Azure, GCP, or any real cloud provider.

It only analyzes local demo JSON files.

## Privacy

Do not commit:

- Real cloud account IDs
- Real ARNs
- Real bucket names
- Real IAM policies
- Real database names
- Real customer or company data
- Access keys or secrets
